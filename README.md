## Introduction
The following is a list of programming languages that every student should be familiar with. These languages have their own advantages and uses, from web development to software engineering and data science. The sections are written by different team members.

## JavaScript

*Written* by *Gilbert Kiprono*

JavaScript is a general-purpose programming language, mainly used for web development. It can be used by developers to build interactive websites, dynamic user interfaces and web applications. JavaScript is executed on the web browser, as well as in server-side applications like Node.js.

## Key Features

- The course is very easy to learn, so it is good for beginners.
- Works with interactive Web pages
- It can be used on the client side in addition to the server side.
- Large community library and vast libraries.

## Python

*Written* by *Timothy*

Python is a high-level, easily read, easily understood programming language. It is extensively used in data science, AI, web development, automation, and scientific computing.

## Key Features

- Plug-and-play syntax that is easy to use.Easy to use syntax, for the beginner.
- Extensive standard library
- Very popular for Data Analysis and Machine Learning
- Strong community support

## Java

*Written* by *Gladwell*

Java is an object-oriented programming language, which is widely adopted to write enterprise applications, for the development of Android and large scale systems. It has a "Write Once Run Anywhere" approach that enables Java programs to execute on various platforms without substantial changes.

## Key Features
- Platform independent: runs on the Java Virtual Machine (JVM)
- Strong security features
- Object-oriented programming support
- Popular for enterprise applications

## Conclusion

Today, some of the most common programming languages are JavaScript, Python, and Java. These are all languages that can be mastered to create a solid base for a software developer and tech career.
